Working with CSS Selectors

CSS Selectors are the way to select the element from group of tags in web page

Types of CSS Selectors-

Simple Selector
Universal Selector
Group Selector
Pseduclass
PseudoElements

Simple Selector-

Elements are selected based on their name, id, class

Simple selector is one of the easiest way to apply css

Universal Selector-

It is used to apply common style to entire web page
